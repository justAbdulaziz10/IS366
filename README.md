# IS 366 — Final Exam Master 🎯

A complete, self-contained study website for the **IS 366 (AI for Decision Support Systems)** final exam at King Saud University.

**356 questions** compiled from: all the lecture slides in the exam scope, this semester's quizzes + midterm, every past quiz/exam, **plus** extra practice questions and full explanations for every item — all in English.

## Features
- **Exam Simulator** — a realistic mock final: 20 True/False + 20 MCQ, randomly drawn, timed, scored, with full review of wrong answers.
- **Study by Topic** — pick topics, length, and mode (Learn = instant answers + explanations, or Test = score at the end).
- **Study Notes** — a condensed, exam-focused summary of every chapter with quick-reference tables and a list of the top traps.
- Works fully offline once loaded. No tracking, no accounts. Mobile-friendly.

## Topics covered
1. Decision Support Framework & Analytics
2. AI Concepts (Chapter 2) — incl. LLMs / Transformers
3. Data Mining (Chapter 4)
4. Machine Learning Fundamentals (incl. KNN, Reinforcement Learning)
5. Text Mining, Sentiment Analysis & Web/Search Mining (Chapter 7)
